# xrwvm Full Stack Developer Capstone

## Project Title
xrwvm Full Stack Developer Capstone - Car Dealership Application

## Description
This project is a full-stack car dealership web application built using Django (backend) and React (frontend).  
It allows users to view dealers, submit reviews, and analyze sentiments of reviews.

## Features
- User login and logout system
- Dealer listing and filtering
- Dealer review submission
- Sentiment analysis of reviews
- REST API integration with Django backend
- React frontend UI

## Tech Stack
- Django
- Django REST Framework
- React.js
- HTML, CSS, Bootstrap
- SQLite / Database backend

## Project Structure
- backend/ → Django server code  
- frontend/ → React UI  
- static/ → HTML, CSS, images  

## API Endpoints
- /djangoapp/login
- /djangoapp/logout
- /djangoapp/dealers
- /djangoapp/get_cars
- /djangoapp/analyze/<review>
